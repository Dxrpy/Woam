# Woam Adventure Game #
Short text-based adventure game coded purely in python :)
Made by Dxrpy

TO PLAY ONE OF THE ENDINGS IN WOAM YOU HAVE TO INSTALL THE 'pygame' MODULE

HOW TO DO SO:

-PyCharm-
File > Settings > Project: example > Project Interpreter
CLICK THE "+" BUTTON AND TYPE "pygame"
INSTALL THE PACKAGE

-CMD-
TYPE "python -m pip install pygame" INTO THE PROMPT
